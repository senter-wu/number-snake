[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)

## 🎯 游戏目的和好处

### 为什么选择数字贪吃蛇？

#### 🌟 专为儿童设计的学习工具
- **3-10岁适龄**：根据儿童认知发展特点设计
- **趣味性强**：用游戏替代枯燥的练习题
- **正向反馈**：保护学习兴趣，激发自信心

#### 🧠 教育价值

| 能力培养 | 具体说明 |
|---------|----------|
| **数学计算能力** | 20以内加减法熟练掌握 |
| **数感培养** | 理解数字大小、分解与组合 |
| **逻辑思维** | 推理哪些数字可以组合成目标 |
| **观察力** | 在多个数字中快速找到正确组合 |
| **反应能力** | 控制蛇躲避障碍，快速做出决策 |

#### 💡 核心优势

1. **无压力学习环境**
   - 撞墙不会死亡，只会害羞地转弯
   - 咬到自己不会结束游戏
   - 可以随时暂停思考

2. **即时学习反馈**
   - 正确时显示等式 "5 + 3 = 8"
   - 错误时提示正确答案
   - 20秒无操作自动提示

3. **个性化设置**
   - 游戏时长可自由选择（1-60分钟）
   - 音效开关可控制

## 🎮 游戏玩法

### 快速入门

```
选择时长 → 控制蛇移动 → 收集两个数字 → 学习加减法
```

### 详细规则

#### 1️⃣ 每轮目标
每轮会随机生成一个 **目标数字**（1-20之间），以及 **运算方式**（加法或减法）。

#### 2️⃣ 收集数字
- 控制蛇移动，碰到数字泡泡自动收集
- 需要收集 **两个** 数字

#### 3️⃣ 成功/失败判定

| 情况 | 结果 | 奖励/惩罚 |
|------|------|----------|
| 运算结果 = 目标 | ✅ 正确 | 蛇变长、得分+1 |
| 运算结果 ≠ 目标 | ❌ 错误 | 蛇变短、得分-1 |

#### 4️⃣ 示例

> 目标：13，运算方式：加法
> 收集到 8 和 5 → 8 + 5 = 13 ✅ 正确！蛇变长！

> 目标：8，运算方式：减法
> 收集到 10 和 3 → 10 - 3 = 7 ❌ 错误！蛇变短！

### 操作按键

| 按键 | 功能 |
|------|------|
| ⬆️⬇️⬅️➡️ 方向键 | 控制蛇的移动方向 |
| W/A/S/D | 同样可以控制方向 |
| 空格键 | 暂停/继续游戏 |
| R键 | 重新开始游戏 |
| M键 | 开关音效 |
| ESC | 退出游戏 |

### 游戏时长选择

- 预设选项：1分钟、3分钟、5分钟、10分钟
- 自定义：1-60分钟任意选择
